# Ask Pixley

On the start screen, you can ask Pixley questions before even opening a folder.

## How to Use It

1. **Attach a file** — Click "Attach File..." to give Pixley context
2. **Or attach a folder** — Hover over a recent folder and click the blue "Ask" pill
3. **Type your question** — What would you like to know?
4. **Press Enter** — Pixley will answer

## What Can You Ask?

With a file attached:
- "Summarize this document"
- "What are the main points?"
- "Explain the technical terms"

With a folder attached:
- "What's in this folder?"
- "Which file talks about X?"
- "How many documents are there?"

## Opening Documents

If you ask about a file and want to read more, Pixley can open it in the browser view where you can continue chatting about it.

---

*Tip: Pixley uses Apple Intelligence, which runs entirely on your device — your files stay private.*
