# Pixley Writer v1.0 Specification

**Status:** SHIPPING
**Date:** 2026-02-01

## Vision

Pixley Writer is a native SwiftUI markdown reader with liquid glass aesthetics. It provides an elegant way to read markdown files generated by AI tools or found anywhere on the file system.

## v1.0 Features (Shipping Today)

### File Browser Sidebar
- Navigate file system to find .md files
- Folders first, then markdown files alphabetically
- Glass buttons for navigation (back, home)
- Hover states with lift animations

### Shelf (Favorites)
- Drag-drop files to add to shelf
- Double-click in browser to add to shelf
- Persists across sessions
- Quick access to frequently used files

### Markdown Editor/Viewer
- Syntax-highlighted markdown display
- Headers, bold, italic, code, links, lists, blockquotes
- Cmd+S to save edits
- Dirty indicator in header
- Auto-save every 30 seconds

### Liquid Glass UI
- Material blur backgrounds
- Gradient highlights for depth
- Spring animations on interactions
- Haptic feedback on key actions
- Accessibility: reduced motion support

### Default App Handler
- Dismissable banner to set as default .md opener
- Registers for markdown file types

## Future Vision (v1.1+)

- **Live file watching** - Detect when files change on disk
- **Diff visualization** - Show changed lines floating in/out
- **Slack-style notifications** - "↑ Content changed above"
- **Scroll position locking** - Stay where you are while content updates

## Technical Notes

- macOS 14+ required
- Universal binary (Intel + Apple Silicon)
- No sandbox (for full file system access)
- ~800 lines of Swift
