# Welcome to Pixley Reader

**Pixley Reader** is a simple app for reading markdown files. If you've ever received a `.md` file and wondered what to do with it — you're in the right place.

## What's a Markdown File?

Markdown is a way to format text using simple symbols:
- `#` makes a heading
- `**bold**` makes **bold** text
- `-` makes a bullet list (like this one)

Many AI tools (like ChatGPT, Claude, and others) write their responses in markdown. Pixley Reader helps you read these files beautifully.

## What Can You Do?

1. **Browse folders** — Open any folder and see all the markdown files inside
2. **Read beautifully** — Markdown is displayed with nice formatting
3. **Ask Pixley** — Have questions about a document? Ask the AI assistant

## Getting Started

Click a folder from the start screen, or use the buttons on the left to open Desktop, Documents, or Downloads.
